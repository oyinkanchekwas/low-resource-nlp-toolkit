# Technical Scope

## Reliable Core

- Deterministic normalisation and tokenisation.
- Canonical emotion label mapping.
- Lexical and script-aware language routing.
- Evaluation reports and confusion matrices.
- CLI, examples, and CI.

## Current Boundaries

- No private datasets are bundled.
- No model weights are committed.
- No API-backed demos require credentials.
- Routing output should not be treated as a person's language community.

## Adding Modules

New benchmarks, model backends, or research workflows need clear notes on data source, licence,
evaluation method, and limitations.
