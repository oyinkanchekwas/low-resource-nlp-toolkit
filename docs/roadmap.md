# Scope Notes

## Reliable Core

- Deterministic normalisation and tokenisation.
- Canonical emotion label mapping.
- Lexical and script-aware language routing.
- Evaluation reports and confusion matrices.
- CLI, examples, and CI.

## Current Boundaries

- No private datasets are bundled.
- No model weights are committed.
- No API-backed demos require credentials.
- No claim is made that routing identifies a person's language community.

## Expansion Criteria

New benchmarks, model backends, or research workflows belong in the project only when the data
source, licence, evaluation method, and limitations can be documented clearly.
