# Roadmap

## Phase 1: Reliable Core

- Deterministic normalisation and tokenisation.
- Canonical emotion label mapping.
- Lexical and script-aware language routing.
- Evaluation reports and confusion matrices.
- CLI, examples, and CI.

## Phase 2: Public Benchmarks

- Add public benchmark fixtures for African language routing.
- Add data cards describing limitations and collection context.
- Add reproducible evaluation scripts.

## Phase 3: Model Backends

- Optional sentence-transformer routing backend.
- Optional Hugging Face text-classification backend.
- Calibration utilities for low-resource class imbalance.

## Phase 4: Research Workflows

- Dataset audit reports.
- Human review sheets for ambiguous language/emotion examples.
- Integration with responsible AI documentation templates.
