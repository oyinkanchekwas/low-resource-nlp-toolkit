# Version 0.1.0 Release Notes

Version 0.1.0 establishes the toolkit as a small public research package for low-resource NLP preparation and evaluation.

## Included

- Source and wheel distributions for Python 3.9, 3.10, and 3.11.
- Text normalisation utilities for noisy multilingual text.
- Lightweight language-routing support.
- Emotion-label mapping utilities.
- Classification evaluation helpers.
- CLI examples that run without private data, credentials, model weights, or external API calls.
- Documentation covering scope, responsible use, contribution guidance, citation metadata, and release history.

## Published Artefacts

- [GitHub release](https://github.com/oyinkanchekwas/low-resource-nlp-toolkit/releases/tag/v0.1.0)
- [PyPI package](https://pypi.org/project/low-resource-nlp-toolkit/0.1.0/)
- [TestPyPI package](https://test.pypi.org/project/low-resource-nlp-toolkit/0.1.0/)

## Scope

This release does not include private datasets, committed model weights, API-backed demos, or claims that language routing can identify a person's ethnicity, nationality, identity, or language community.
